This is a POC (proof of concept) version of the Vivit Mobile app. It is not what the actual Vivit Mobile app will look like. It is purely for experimentation with various design ideas and the technologies required to create the eventual real Vivit mobile apps

Items that work right now:
  1. Login/Join Vivit Page - the basic UI elements are in place but no API calls are being made yet - so it doesn't do anything for now.

  2. Logout Page - a UI element is in place but no API calls are being made yet.

  3. Upcoming Events Page - events display live from the Vivit website. Click on the event title to get more info. Click on Register link to go to the Vivit website to register for that event.

  4. Vivit Website Page - Opens a browser with the Vivit website.
